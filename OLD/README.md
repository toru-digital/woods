# woods

Explore the woods

https://toru-digital.github.io/woods/
http://localhost:5173/woods/

## Usage

```bash
nvm use 18;
npm i;
npx vite serve;
```

## Deploy

Check the build out locally first...

```bash
npx vite build;
npx vite preview;
```

Push to the live branch, Github pages will depoloy automatically
